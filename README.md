# Gestor de biblioteca

## ¿Que es el proyecto?
Una aplicación de consola para gestionar una biblioteca, sus libros/socios y sistemas de prestar/devolver libros.

## ¿Que hace?
- Agregar libros y socios a una biblioteca determinada.
- Prestar libros a un socio determinado (con limite de 3 libros por socio).
- Que un socio devuelva un libro.
- Listar los libros y los libros disponibles para prestar.
- Ver los libros en posesión de un socio determinado.

## Tecnologías usadas.
Se utiliza únicamente python.
Se usan librerías internas de python, no hay dependencias externas.

## Como instalarlo y ejecutarlo.
Como no hay dependencias de ningún tipo mas allá de tener python instalado, simplemente hay que ejecutar el archivo main.py desde la carpeta src.
```
cd src
python main.py
```
